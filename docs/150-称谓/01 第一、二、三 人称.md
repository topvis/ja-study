::: info 💡 日语人称核心避坑指南

在实际日语交流中，**频繁使用第二人称代词（特别是“あなた”）会显得非常失礼或产生强烈的距离感**。最地道、最礼貌的做法是：**直接称呼对方的【姓氏/名字 + さん】或【职位】**（例如：田中さん、部長）。只有在完全不知道对方名字，或者填写问卷、夫妻/情侣吵架等极特定场景下才会使用第二人称代词。

:::

### 日语人称代词常用法速查表

#### 一、 第一人称（我）

| 代词       | 读音     | 适用性别 | 语气与使用场景提示                                                                                            |
| :--------- | :------- | :------- | :------------------------------------------------------------------------------------------------------------ |
| **私**     | わたし   | 男女通用 | **最通用、最保险的自称**。男性在职场/正式场合必用；女性在日常和正式场合均可用。                               |
| **私**     | わたくし | 男女通用 | **极度正式、恭敬**。用于商务谈判、面对高层客户、发表正式演讲或服务业对顾客。                                  |
| **僕**     | ぼく     | 主要男性 | **温和、谦逊、无攻击性**。常用于小男孩、年轻男性，或中老年男性在非正式职场、朋友间的温和自称。                |
| **俺**     | おれ     | 男性     | **随性、粗犷、有男子气概**。男性在私下（朋友、家人、下属面前）最常用的自称。**绝对不能在正式/商务场合使用**。 |
| **あたし** | あたし   | 女性     | **女性化、娇柔、口语化**。由「わたし」演变而来，常用于年轻女性或语气较柔和的日常私下交流。                    |
| **自分**   | じぶん   | 男女通用 | **客观、带有一点体育系/军人口吻**。年轻人（尤其是运动健儿）或职场新人常用，听起来比较认真、守纪律。           |

---

#### 二、 第二人称（你）

| 代词       | 读音            | 适用性别 | 语气与使用场景提示                                                                                                |
| :--------- | :-------------- | :------- | :---------------------------------------------------------------------------------------------------------------- |
| **あなた** | あなた          | 男女通用 | **教科书标准“你”**。但现实中对上级用不礼貌。多用于：①不知道对方名字的问卷/广告；②**妻子对丈夫的称呼（亲爱的）**。 |
| **君**     | きみ            | 男女通用 | **上对下的关怀/亲近**。多为男性长辈、上司对晚辈、下属的称呼，或学校里男生对熟悉同学的称呼。                       |
| **お前**   | おまえ          | 主要男性 | **粗鲁、极其亲近、大男子主义**。男性对哥们、晚辈，或丈夫对妻子用。对不熟的人用带有挑衅意味。                      |
| **あんた** | あんた          | 男女通用 | **「あなた」的音变，口语、市井**。带有一点抱怨、随便或亲昵的语气，多用于熟人、家人之间（动漫中傲娇角色常用）。    |
| **手前**   | てまえ / てめえ | 主要男性 | **极其粗鲁、带有极强的敌意**。通常读作 _てめえ_，属于打架或动漫里骂人“你这混蛋”的用语，日常千万别用。             |

---

#### 三、 第三人称（他/她/他们）

_注：与第二人称类似，现实中更倾向于用「田中さん」或「あの方」来指代第三者。_

| 代词         | 读音         | 释义         | 语气与使用场景提示                                                                      |
| :----------- | :----------- | :----------- | :-------------------------------------------------------------------------------------- |
| **彼**       | かれ         | 他           | **普通的“他”**。在恋爱语境下，也可以直接表示**“男朋友”**（例如：彼氏 / かれし）。       |
| **彼女**     | かのじょ     | 她           | **普通的“她”**。在恋爱语境下，也可以直接表示**“女朋友”**。                              |
| **あの方**   | あのかた     | 那位（尊敬） | **最礼貌的第三人称**。在商务或正式场合，指代不在场的尊贵客人或上级时使用。              |
| **あの人**   | あのひと     | 那个人       | **中性、普通的指代**。日常生活中提起某个不熟或不在场的人时使用。                        |
| **あいつ**   | あいつ       | 那家伙       | **随便、口语化**。指代同龄人、晚辈或讨厌的人。带有“那家伙/那小子”的意思，不能对长辈用。 |
| **彼ら**     | かれら       | 他们         | **复数形式**。用来指代一群人（通常包含男性，或全为男性）。                              |
| **彼女たち** | かのじょたち | 她们         | **复数形式**。专门用来指代全为女性的第三者群体。                                        |
