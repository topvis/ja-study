### 日语家庭成员称谓对照表（含叔伯姑舅姨及亲昵称呼）

| 关系                    | 称呼自己的家人 (谦称/内)      | 称呼别人的家人/在家叫长辈 (尊称/外)           | 中文释义与使用场景提示                                                                                                                                                                                     |
| :---------------------- | :---------------------------- | :-------------------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **父亲**                | 父 (ちち)                     | お父さん (おとうさん) / パパ                  | **ちち**：对人提起自己父亲时用。<br>**お父さん**：在家叫爸爸，或称呼对方父亲。<br>**パパ**：幼儿时期或日常非常亲昵、口语化的称呼。                                                                         |
| **母亲**                | 母 (はは)                     | お母さん (おかあさん) / ママ                  | **はは**：对人提起自己母亲时用。<br>**お母さん**：在家叫妈妈，或称呼对方母亲。<br>**ママ**：幼儿时期或日常非常亲昵、口语化的称呼。                                                                         |
| **叔叔/伯父/舅舅**      | 叔父 / 伯父 (おじ)            | おじさん                                      | **伯父**指比父母年长者，**叔父**指比父母年轻者（对人外起时统称 **おじ**）。<br>在家里称呼自己的叔伯舅，或称呼路人中年男子时用 **おじさん**（或更亲昵的 _おじちゃん_）。称呼别人的叔伯舅用 **おじさん**。   |
| **姑姑/姨妈/婶婶/舅妈** | 叔母 / 伯母 (おば)            | おばさん                                      | **伯母**指比父母年长者，**叔母**指比父母年轻者（对人提起时统称 **おば**）。<br>在家里称呼自己的姑姨婶舅妈，或称呼路人中年女子时用 **おばさん**（或更亲昵的 _おばちゃん_）。称呼别人的姑姨用 **おばさん**。 |
| **丈夫**                | 夫 (おっと) / 主人 (しゅじん) | 旦那さん (だんなさん) / ご主人 (ごしゅじん)   | 介绍自己丈夫常用 **夫** 或 **主人**。称呼对方丈夫用 **ご主人** 或 **旦那さん**。                                                                                                                           |
| **妻子**                | 妻 (つま) / 家内 (かない)     | 奥さん (おくさん)                             | 介绍自己妻子常用 **妻**。称呼对方妻子绝对要用 **奥さん**。                                                                                                                                                 |
| **哥哥**                | 兄 (あに)                     | お兄さん (おにいさん)                         | 在家叫自己哥哥时用 **お兄さん**（或更亲昵的 _お兄ちゃん_）。对别人提起时用 **兄**。                                                                                                                        |
| **姐姐**                | 姉 (あね)                     | お姉さん (おねえさん)                         | 在家叫自己姐姐时用 **お姉さん**（或更亲昵的 _お姉ちゃん_）。对别人提起时用 **姉**。                                                                                                                        |
| **弟弟**                | 弟 (おとうと)                 | 弟さん (おとうとさん)                         | 自己的弟弟直接叫名字或称呼 **弟**。称呼别人的弟弟要加 **さん**。                                                                                                                                           |
| **妹妹**                | 妹 (いもうと)                 | 妹さん (いもうとさん)                         | 自己的妹妹直接叫名字或称呼 **妹**。称呼别人的妹妹要加 **さん**。                                                                                                                                           |
| **祖父**                | 祖父 (そふ)                   | おじいさん                                    | 在家叫自己爷爷/外公用 **おじいさん**（或 _おじいちゃん_）。对别人提起用 **祖父**。                                                                                                                         |
| **祖母**                | 祖母 (そぼ)                   | おばあさん                                    | 在家叫自己奶奶/外婆用 **おばあさん**（或 _おばあちゃん_）。对别人提起用 **祖母**。                                                                                                                         |
| **儿子**                | 息子 (むすこ)                 | 息子さん (むすこさん)                         | 称呼对方儿子要加 **さん**。                                                                                                                                                                                |
| **女儿**                | 娘 (むすめ)                   | 娘さん (むすめさん) / お嬢さん (おじょうさん) | 称呼对方女儿可用 **娘さん**，非常礼貌客气的场合可以用 **お嬢さん**。                                                                                                                                       |
